# SST Rekognition

### An event-driven app that uses AWS Rekognition to labeling images

### Including

- API Gateway
- DynamoDB
- EventBridge / Event Bus
- Lambda Functions
- Rekognition
- Simple Queue Service (SQS)
- Simple Storage Service (S3)
- Vite Static Site
