# SST Rekognition

### An event-driven app to label images on top of AWS

### Including

- API Gateway
- DynamoDB
- EventBridge / Event Bus
- Lambda Functions
- Rekognition
- Simple Queue Service (SQS)
- Simple Storage Service (S3)
- Vite Static Site
